# Vexter_PoE_Modules

Small SMD PoE modules that comply with IEEE 802.3af standard
parameters:
  + IEEE 802.3af-compliant
  + two 12W variants: 5V (2.4A) and 12V(1A)
  + Small SMD footprint (3x1.5cm)
